# Visualisation Sismique Algérie

## Description

Cette application web interactive permet de visualiser les données sismiques de l'Algérie, en mettant en évidence les wilayas et communes selon leur classification sismique. L'interface utilise une carte interactive pour présenter les informations de manière claire et accessible.

## Fonctionnalités

- **Carte interactive** : Visualisation des wilayas et communes algériennes avec classification sismique
- **Filtres par zone sismique** : Possibilité de filtrer les données par niveau de risque (Zone 0, I, IIa, IIb, III)
- **Indicateurs clés** : Affichage des statistiques principales (nombre de wilayas, communes, etc.)
- **Interface responsive** : Adaptée aux différents appareils et tailles d'écran

## Technologies utilisées

- **HTML5/CSS3** : Structure et mise en page
- **JavaScript** : Logique interactive
- **Leaflet** : Bibliothèque de cartographie
- **Données GeoJSON** : Géométries des wilayas et communes

## Installation et utilisation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/LotfiDjebbar/visualisation_seisme.git
   ```

2. Ouvrir le fichier `seisme.html` dans un navigateur web moderne

Aucune installation supplémentaire n'est requise - l'application fonctionne directement dans le navigateur.
